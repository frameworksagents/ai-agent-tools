# Matrice décision agent vs automatisation

Toutes les tâches ne nécessitent pas un agent IA.

## Utiliser un workflow si

- le processus est stable
- les étapes sont connues
- la logique est déterministe

## Utiliser un agent si

- le problème nécessite du raisonnement
- le contexte est variable
- les décisions doivent être dynamiques

## Architecture hybride

Dans de nombreux cas, la meilleure approche combine :

workflow + agent IA
