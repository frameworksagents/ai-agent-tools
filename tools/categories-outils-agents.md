# Catégories d’outils pour agents IA

Construire un système d’agents IA nécessite généralement plusieurs types d’outils.

## Modèles de langage

Les modèles de langage fournissent la capacité de raisonnement et de génération de texte.

Ils constituent le cœur du système.

## Outils externes

Les agents peuvent utiliser des outils pour interagir avec des systèmes externes :

- API
- bases de données
- moteurs de recherche
- services internes

## Mémoire

La mémoire permet à un agent de conserver des informations entre différentes étapes.

Elle peut être :

- temporaire
- persistante
- basée sur un système de récupération d’information

## Orchestration

L’orchestration coordonne les actions des agents et l’exécution des tâches.

Elle gère :

- les workflows
- les interactions entre agents
- les appels d’outils
