# Checklist sécurité pour agents IA

Les systèmes d’agents doivent être conçus avec des mécanismes de sécurité adaptés.

## Gestion des accès

- limiter les permissions
- contrôler les accès aux API

## Validation des entrées

- filtrer les entrées utilisateur
- éviter les injections de prompt

## Limitation des actions

- restreindre les actions possibles
- valider les actions critiques

## Journalisation

- enregistrer les actions importantes
- surveiller les comportements anormaux
